# 干瞪眼 (Gan Deng Yan) - WeChat Mini Program

一款经典的中国扑克牌游戏微信小程序实现。"干瞪眼"因其简单易学、节奏快的特点，成为了众多玩家喜爱的纸牌游戏。

## 🎮 游戏特点

- **2-5人游戏**：支持2到5个玩家参与
- **完整规则实现**：包含所有基础和特殊牌型
- **简洁UI设计**：适配所有屏幕尺寸
- **流畅游戏体验**：快速的游戏流程和直观的操作

## 📋 游戏规则

### 基础设定

| 项目 | 说明 |
|------|------|
| 牌具 | 标准54张扑克牌（含大小王），不区分花色，只比点数 |
| 点数大小 | 2 > A > K > Q > J > 10 > 9 > 8 > 7 > 6 > 5 > 4 > 3 |
| 游戏人数 | 2-5人可选 |
| 庄家规则 | 庄家初始6张牌，其他玩家5张牌 |

### 核心牌型

#### 基础牌型（必须按"大一法则"接牌）

| 牌型 | 构成 | 接牌规则 |
|------|------|--------|
| 单张 | 1张牌 | 只能出点数大1的单牌 |
| 对子 | 2张相同点数牌 | 只能出点数大1的对子 |
| 三张 | 3张相同点数牌 | 只能出点数大1的三张 |
| 顺子 | ≥3张连续点数牌（不含2） | 只能出点数大1的同长度顺子 |
| 连对 | ≥2对连续点数牌（不含2） | 只能出点数大1的同长度连对 |

#### 特殊牌型（可突破大一法则）

| 牌型 | 构成 | 功能 |
|------|------|------|
| 炸弹 | 4张相同点数牌 | 可接任何基础牌型，也可压过比自己小的炸弹 |
| 三带王 | 3张相同点数牌 + 1张王 | 可接同类型基础牌型，压过普通三张 |
| 王 | 大小王 | 万能牌，可代替任何一张牌，但不能单独出牌 |

### 游戏流程

1. **发牌阶段**：庄家先发牌，庄家6张，其他玩家5张
2. **出牌阶段**：庄家首先出任意符合规则的牌型
3. **接牌/过牌**：轮流出牌，按"大一法则"接牌，无法接牌则过牌
4. **补牌机制**：所有玩家都过牌后，最后出牌的玩家补1张底牌
5. **报单**：手中仅剩1张牌时，必须主动喊"报单"

### 胜负判定

- **胜利条件**：最先将手中所有牌出完的玩家获胜
- **计分规则**：
  - 剩余牌数为分数（剩1张不计分）
  - 胜者得所有负者分数之和
  - 炸弹记分翻倍

## 🚀 快速开始

### 开发环境要求

- 微信开发者工具（最新版本）
- Node.js 12+ (可选，用于项目管理)

### 安装与运行

1. **克隆或下载项目**
   ```bash
   git clone https://github.com/huanglw812/dengyan.git
   cd dengyan
   ```

2. **打开微信开发者工具**
   - 选择"小程序项目"
   - 导入此项目文件夹
   - 使用测试AppID

3. **编译与预览**
   - 点击"编译"按钮
   - 在编辑器或手机上预览游戏
